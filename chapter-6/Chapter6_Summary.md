# บทที่ 6 — รีไซเคิลขยะ
### (Recycling the Garbage)
*Why We Die — Venki Ramakrishnan*

---

## แก่นของบท (One-sentence takeaway)
> **ร่างกายเราผลิตโปรตีนมากกว่า 50 ล้านคนกำลังทำงานทุกวินาทีในแต่ละเซลล์ โปรตีนเหล่านี้ต้องพับตัวเป็นรูปร่าง 3 มิติที่ถูกต้องจึงจะทำงานได้ — เมื่อพับผิด ร่างกายมีระบบ "รีไซเคิล" เพื่อกำจัดขยะ แต่เมื่ออายุมากขึ้น ระบบนี้เสื่อมลง ส่งผลให้เกิดโรคร้ายแรงอย่าง อัลไซเมอร์ พาร์กินสัน และโรคสมองเสื่อมอื่นๆ**

## 🧠 ภาพเปิดเรื่อง: ขยะที่สะสมทุกวัน

ทุกวันเมื่อเราลืมหนังสือ วางถุงมือ ร่ม หรือหมวกไว้ตามที่ต่างๆ ของเหล่านั้นอาจกลายเป็นสัญญาณของความเสื่อมไปตามอายุ ในวัย 70 ปี ความเสื่อมที่สะสมเริ่มส่งผลรุนแรง — ไม่ใช่แค่ของหาย แต่สมองและร่างกายก็เริ่มสะสม "ขยะ" โมเลกุลที่ทำงานผิดปกติ

**ขนาดของปัญหา:**
- ทุกวันนี้ผู้คนมากกว่า **50 ล้านคน** กำลังทำงานทุกขณะในเซลล์ของเรา
- ในเกือบทุกประเทศทั่วโลก คาดว่าตัวเลขจะเพิ่มเป็น **78 ล้านคน** ภายในปี 2030 และ **139 ล้านคน** ภายในปี 2050
- ในอังกฤษ ภาวะสมองเสื่อมเป็นสาเหตุการตายอันดับหนึ่ง

> 🧠 **วิธีจำ:** เซลล์เหมือน "โรงงาน" ขนาดใหญ่ — ถ้าระบบกำจัดขยะพัง ขยะก็สะสมจนโรงงานทำงานไม่ได้

## 🔬 1. โปรตีน — เครื่องจักรแห่งชีวิต

### โปรตีนคืออะไร?
โปรตีนเป็นเครื่องจักรระดับโมเลกุลที่ทำงานแทบทุกอย่างในเซลล์ — ตั้งแต่เร่งปฏิกิริยาเคมี ส่งสัญญาณ สร้างโครงสร้าง ไปจนถึงป้องกันร่างกาย

### การพับตัวของโปรตีน (Protein Folding)
- โปรตีนถูกสร้างเป็น **สายยาว** ของกรดอะมิโน
- สายนี้ต้อง **พับตัว** เป็นรูปร่าง 3 มิติที่ถูกต้อง จึงจะทำงานได้
- การพับตัวเป็นกระบวนการที่ซับซ้อนมาก — โปรตีนหนึ่งตัวอาจพับตัวได้เป็น **ล้านๆ รูปแบบ** แต่มีเพียงรูปแบบเดียวที่ถูกต้อง
- **Chaperone proteins** (โปรตีนพี่เลี้ยง) ช่วยในการพับตัวให้ถูกต้อง

### ปัญหาเมื่อพับผิด
- โปรตีนที่พับผิดจะ **ไม่ทำงาน** หรือทำงานผิดปกติ
- บางครั้งโปรตีนที่พับผิดจะ **จับกลุ่มกัน** เป็นก้อน (aggregate) ที่เป็นพิษต่อเซลล์
- เมื่ออายุมากขึ้น กระบวนการพับตัวผิดพลาดมากขึ้น

> 💡 **เปรียบเทียบ:** โปรตีนเหมือน **กระดาษ origami** — ต้องพับตามขั้นตอนที่ถูกต้องทุกจุดจึงจะได้รูปร่างสมบูรณ์ ถ้าพับผิดแม้จุดเดียว ผลลัพธ์จะผิดรูปทั้งหมด

## 🔄 2. ระบบควบคุมคุณภาพโปรตีน — Proteasome & Ubiquitin

### ยูบิควิติน (Ubiquitin) — "สติกเกอร์รีไซเคิล"
- **Ubiquitin** เป็นโปรตีนขนาดเล็กที่พบในสิ่งมีชีวิตเกือบทุกชนิด (จึงได้ชื่อว่า "ubiquitous" = มีอยู่ทุกที่)
- ถูกค้นพบในช่วงทศวรรษ **1970** โดยนักวิทยาศาสตร์ชาวอิสราเอล
- ทำหน้าที่เป็น **"สติกเกอร์"** ติดบนโปรตีนที่เสียหาย เพื่อส่งสัญญาณว่า "ตัวนี้ต้องถูกกำจัด"

### โปรตีเอโซม (Proteasome) — "เครื่องบดขยะ"
- เป็นเครื่องจักรโมเลกุลรูป **ทรงกระบอก** ที่ทำหน้าที่ย่อยสลายโปรตีน
- เมื่อโปรตีนที่ติดป้ายยูบิควิตินเข้าไปในโปรตีเอโซม จะถูก **ย่อยเป็นชิ้นเล็กๆ**
- ชิ้นส่วนที่ได้สามารถ **นำกลับมาใช้ใหม่** เพื่อสร้างโปรตีนใหม่
- ภาพเปรียบว่าเป็น **"เครื่องย่อยสลาย"** ที่ย่อยชิ้นส่วนเก่าแล้วนำกลับมาใช้ใหม่

### ปัญหาเมื่ออายุมากขึ้น
- ระบบยูบิควิติน-โปรตีเอโซมทำงาน **ลดลง** ตามอายุ
- โปรตีนที่เสียหายสะสมมากขึ้น → นำไปสู่โรคต่างๆ
- เชื่อมโยงกับโรค **อัลไซเมอร์ โรคพาร์กินสัน** และอื่นๆ

> 🧠 **วิธีจำ:** Ubiquitin = "สติกเกอร์ถังขยะ" 🏷️ → ติดบนโปรตีนเสีย → Proteasome = "เครื่องบดขยะ" 🗑️ → บดแล้วรีไซเคิล

## 🧫 3. ไลโซโซมและออโทฟาจี — ระบบกำจัดขยะชุดที่สอง

### Christian de Duve — ผู้ค้นพบ "กระเพาะของเซลล์"
- ในปี **1955** คริสเตียน เดอ ดูเว (Christian de Duve) ศาสตราจารย์ชาวเบลเยียมแห่งมหาวิทยาลัยคาทอลิกแห่งลูแวน ค้นพบ **ไลโซโซม (Lysosome)**
- ไลโซโซมเป็น **"กระเพาะ"** ของเซลล์ — ถุงที่บรรจุเอนไซม์ย่อยสลาย
- ทำหน้าที่กลืนและย่อยวัตถุขนาดใหญ่ รวมถึง **ออร์แกเนลล์** ที่เสียหาย

### ออโทฟาจี (Autophagy) — "เซลล์กินตัวเอง"
- De Duve ยังค้นพบ **autophagy** = กระบวนการที่เซลล์ "กินตัวเอง"
- เซลล์สร้าง **ออโทฟาโกโซม (autophagosome)** — ถุงที่ห่อหุ้มชิ้นส่วนเสียหาย
- นำไปรวมกับไลโซโซม → ย่อยสลาย → นำกลับมาใช้ใหม่

### Yoshinori Ohsumi — ผู้ไขปริศนาออโทฟาจี
- ในช่วงทศวรรษ **1980–1990** โยชิโนริ โอสุมิ (Yoshinori Ohsumi) ศึกษากระบวนการออโทฟาจีในยีสต์
- ค้นพบ **ยีนสำคัญ** ที่ควบคุมกระบวนการออโทฟาจี
- 🏆 **รางวัลโนเบลสาขาสรีรวิทยาหรือการแพทย์ ปี 2016**

### ออโทฟาจีกับความชรา
- เมื่ออายุมากขึ้น กระบวนการออโทฟาจี **ลดประสิทธิภาพ**
- "ขยะ" สะสมในเซลล์ → เซลล์ทำงานแย่ลง → นำไปสู่ความชรา
- การกระตุ้นออโทฟาจีอาจช่วย **ชะลอวัย** ได้ — เหมือนการ "ทำความสะอาดบ้าน" อย่างสม่ำเสมอ

> 💡 **เปรียบเทียบ:** ถ้า Proteasome คือ "เครื่องบดขยะ" → Lysosome คือ "เตาเผาขยะ" ที่จัดการขยะชิ้นใหญ่กว่า ส่วน Autophagy คือ "รถเก็บขยะ" ที่ออกเก็บชิ้นส่วนเสียหายทั่วเซลล์

## 🍬 4. ไกลโคซิเลชันและไกลเคชัน — น้ำตาลทำลายโปรตีน

### Glycosylation vs. Glycation
- **Glycosylation** (ไกลโคซิเลชัน) — กระบวนการ **ปกติ** ที่เซลล์ติดน้ำตาลบนโปรตีน เพื่อช่วยการทำงาน
- **Glycation** (ไกลเคชัน) — กระบวนการ **ผิดปกติ** ที่น้ำตาลจับกับโปรตีนโดยไม่มีการควบคุม
- ไกลเคชันเพิ่มขึ้นเมื่ออายุมากขึ้น → ทำให้โปรตีนเปลี่ยนแปลงและทำงานผิดปกติ

### ผลกระทบ
- ไกลเคชันเป็นสาเหตุของ **ปัญหาสุขภาพ** หลายอย่าง: โรคเบาหวาน โรคหัวใจ และโรคสมองเสื่อม
- น้ำตาลในเลือดสูง → ไกลเคชันเพิ่มขึ้น → โปรตีนเสียหายมากขึ้น

## ⚠️ 5. Unfolded Protein Response (UPR) & Integrated Stress Response (ISR)

### เมื่อโปรตีนพับผิดมากเกินไป
- เซลล์มีระบบเตือนภัยที่เรียกว่า **Unfolded Protein Response (UPR)** 
- เมื่อเซลล์ตรวจพบว่ามีโปรตีนพับผิดมากผิดปกติ → เซลล์จะ:
  1. **ชะลอ** การผลิตโปรตีนใหม่
  2. **เพิ่ม** การผลิต chaperone เพื่อช่วยพับโปรตีน
  3. **เร่ง** การกำจัดโปรตีนที่พับผิด

### Integrated Stress Response (ISR)
- **ISR** เป็นระบบตอบสนองต่อความเครียดที่กว้างกว่า UPR
- จัดการกับปัญหาหลายอย่าง: โปรตีนพับผิด การติดเชื้อไวรัส ภาวะขาดสารอาหาร
- ISR ทำให้เซลล์ **ลดการผลิตโปรตีนทั่วไป** แต่ **เพิ่มการผลิตโปรตีนเฉพาะ** ที่ช่วยรับมือกับความเครียด

### ปัญหาของ ISR กับความชรา
- เมื่ออายุมาก ISR อาจ **ทำงานมากเกินไป** หรือ **ทำงานผิดปกติ**
- ISR ที่ทำงานเกินควร → ลดการผลิตโปรตีนมากจนเซลล์ขาดโปรตีนที่จำเป็น
- โดยเฉพาะในสมอง — ISR ที่ผิดปกติเชื่อมโยงกับ **โรคสมองเสื่อม**

---

## 💊 6. ISRIB — ความหวังใหม่ในการต่อสู้กับความชรา

### การค้นพบ ISRIB
- **ISRIB** (Integrated Stress Response Inhibitor) คือสารที่ยับยั้ง ISR
- ค้นพบโดยทีมวิจัยของ **นาฮุม โซเนนเบิร์ก (Nahum Sonenberg)** แห่งมหาวิทยาลัยแมกกิลล์ และ **ปีเตอร์ วอลเตอร์ (Peter Walter)** แห่งมหาวิทยาลัยแคลิฟอร์เนีย ซานฟรานซิสโก

### ผลการทดลอง
- ISRIB ได้รับการอธิบายว่าเป็น "**โมเลกุลมหัศจรรย์**" 
- ในหนูทดลอง: ช่วยฟื้นฟูความจำและการเรียนรู้ในหนูแก่
- บริษัทเทคชีวภาพหลายแห่งกำลังพัฒนา ISRIB เพื่อใช้ทางการแพทย์

### ข้อควรระวัง
- ISR มากเกินไป = เซลล์ขาดโปรตีน แต่ ISR น้อยเกินไป = โปรตีนเสียสะสม
- ต้องหาจุดสมดุลที่เหมาะสม — ไม่ใช่แค่ "ปิด" หรือ "เปิด" ISR ทั้งหมด

## 🧟 7. โปรตีนนอกรีต — ไพรออนและโรคสมองเสื่อม

### Carleton Gajdusek — นักล่าโรคปริศนา
- **คาร์ลตัน กายดูเชก (Carleton Gajdusek)** นักวิจัยที่สถาบันสุขภาพแห่งชาติ (NIH) สหรัฐอเมริกา
- ศึกษาโรค **คูรู (Kuru)** ในชนเผ่าฟอร์ (Fore) ในปาปัวนิวกินี
- คูรูเป็นโรคสมองเสื่อมร้ายแรง แพร่ผ่านพิธีกรรม **กินเนื้อคน** (cannibalism)
- กายดูเชกพบว่าโรคนี้ติดต่อได้ → 🏆 **รางวัลโนเบล**

### Stanley Prusiner — ผู้ค้นพบไพรออน
- **สแตนลีย์ พรูซิเนอร์ (Stanley Prusiner)** จากมหาวิทยาลัยแคลิฟอร์เนีย ซานฟรานซิสโก
- ศึกษาโรค **สเครปี (Scrapie)** ในแกะ ซึ่งเป็นโรคสมองเสื่อมคล้ายคูรู
- ค้นพบว่าสาเหตุไม่ใช่ไวรัสหรือแบคทีเรีย แต่เป็น **โปรตีนที่พับตัวผิดปกติ** — เรียกว่า **ไพรออน (Prion)**
- ไพรออนสามารถ "**ชักชวน**" โปรตีนปกติให้พับตัวผิดตาม → แพร่กระจายเหมือนโดมิโน
- 🏆 **รางวัลโนเบลสาขาสรีรวิทยาหรือการแพทย์ ปี 1997**

### โรคที่เกี่ยวข้องกับไพรออน
- **Creutzfeldt-Jakob Disease (CJD)** — โรคสมองฟองน้ำในมนุษย์
- **Bovine Spongiform Encephalopathy (BSE)** — โรควัวบ้า
- ประมาณ **10%** ของผู้ป่วยเกิดจากกรรมพันธุ์ ที่เหลือเกิดจากการติดเชื้อหรือเกิดขึ้นเอง

> 🤯 **ข้อเท็จจริงชวนตกใจ:** ไพรออนไม่มี DNA หรือ RNA — มันเป็นแค่โปรตีนที่พับผิด แต่สามารถ "ติดต่อ" และ "แพร่พันธุ์" ได้โดยเปลี่ยนโปรตีนปกติให้กลายเป็นไพรออนตาม!

## 🧩 8. อัลไซเมอร์ — โรคแห่งโปรตีนพับผิด

### Alois Alzheimer — ผู้ค้นพบโรค
- **อลอยส์ อัลไซเมอร์ (Alois Alzheimer)** แพทย์ชาวเยอรมัน รายงานกรณีแรกในปี **1906**
- ผู้ป่วยชื่อ Auguste Deter — มีอาการสูญเสียความจำ สับสน พฤติกรรมเปลี่ยน
- หลังเสียชีวิต พบ **คราบ (plaques)** และ **ปม (tangles)** ในสมอง

### กลไกของโรค
- **Amyloid-beta (Aβ)** — โปรตีนชิ้นเล็กที่ตัดมาจาก **Amyloid Precursor Protein (APP)**
  - Aβ จับกลุ่มกันเป็น **คราบแอมีลอยด์** นอกเซลล์ประสาท
- **โปรตีนเทา (Tau)** — โปรตีนที่ปกติช่วยรักษาโครงสร้างของ **axon** (แกนประสาท)
  - เมื่อผิดปกติ → tau จับกลุ่มเป็น **ปม (tangles)** ภายในเซลล์ประสาท
- **Presenilin** — เอนไซม์ที่ตัด APP → เมื่อกลายพันธุ์ → ผลิต Aβ มากผิดปกติ

### ข้อถกเถียง: Amyloid Hypothesis
- **สมมติฐานแอมีลอยด์** — เชื่อว่าการสะสมของ Aβ เป็นสาเหตุหลักของอัลไซเมอร์
- แต่ยา **ที่กำจัด Aβ ยังไม่ประสบความสำเร็จ** ในการรักษาอย่างชัดเจน
- Aβ อาจเป็นแค่ **อาการ** ไม่ใช่สาเหตุที่แท้จริง
- ปัจจุบันนักวิทยาศาสตร์กำลังพิจารณาว่า tau, neuroinflammation และกลไกอื่นอาจสำคัญกว่า

### ข้อเท็จจริงที่น่าตกใจ
- การรักษาด้วยยาต้าน amyloid เช่น **Lecanemab** ลดการเสื่อมของสมองได้เพียง **25%** ในช่วง 18 เดือน
- ค่าใช้จ่ายสูงมาก และมีผลข้างเคียงร้ายแรง เช่น อาการบวมและเลือดออกในสมอง

## 🔗 9. ภาพรวม: ทำไมเรื่อง "ขยะโปรตีน" จึงเกี่ยวกับความตาย

### วงจรแห่งความเสื่อม
1. เมื่ออายุมากขึ้น → โปรตีนพับผิดมากขึ้น
2. ระบบกำจัดขยะ (Proteasome, Lysosome, Autophagy) เสื่อมลง
3. โปรตีนเสียสะสม → เซลล์ทำงานผิดปกติ
4. ในสมอง: amyloid plaques + tau tangles → สมองเสื่อม
5. ISR ทำงานผิดปกติ → เซลล์ผลิตโปรตีนไม่ได้ตามปกติ

### ความหวัง
- **ISRIB** และสารยับยั้ง ISR อื่นๆ อาจช่วยฟื้นฟูการทำงานของเซลล์แก่
- **การกระตุ้น autophagy** อาจช่วยทำความสะอาดขยะโปรตีน
- นักวิทยาศาสตร์กำลังพัฒนายาที่ช่วยให้ระบบควบคุมคุณภาพโปรตีนทำงานดีขึ้น
- เป้าหมายสูงสุด: ทำให้เซลล์แก่กลับมา **จัดการขยะได้ดีเหมือนเซลล์หนุ่มสาว**

## 📖 คำศัพท์สำคัญที่ควรจำ

| คำศัพท์ | ความหมาย |
|---|---|
| **Protein Folding** | กระบวนการพับตัวของโปรตีนเป็นรูปร่าง 3 มิติที่ถูกต้อง |
| **Chaperone Protein** | โปรตีนพี่เลี้ยงที่ช่วยให้โปรตีนอื่นพับตัวถูกต้อง |
| **Ubiquitin** | โปรตีนขนาดเล็กที่ "ติดป้าย" โปรตีนเสียหายเพื่อส่งไปกำจัด |
| **Proteasome** | เครื่องจักรโมเลกุลรูปทรงกระบอกที่ย่อยสลายโปรตีนที่ติดป้ายยูบิควิติน |
| **Lysosome** | "กระเพาะ" ของเซลล์ — ถุงที่บรรจุเอนไซม์ย่อยสลาย (ค้นพบโดย Christian de Duve) |
| **Autophagy** | กระบวนการ "เซลล์กินตัวเอง" — กำจัดชิ้นส่วนเสียหายภายในเซลล์ |
| **Autophagosome** | ถุงที่ห่อหุ้มชิ้นส่วนเสียหายก่อนนำไปรวมกับไลโซโซม |
| **Glycosylation** | กระบวนการปกติที่เซลล์ติดน้ำตาลบนโปรตีน |
| **Glycation** | กระบวนการผิดปกติที่น้ำตาลจับกับโปรตีนโดยไม่มีการควบคุม |
| **UPR (Unfolded Protein Response)** | ระบบเตือนภัยเมื่อเซลล์มีโปรตีนพับผิดมากเกินไป |
| **ISR (Integrated Stress Response)** | ระบบตอบสนองต่อความเครียดหลายรูปแบบในเซลล์ |
| **ISRIB** | สารยับยั้ง ISR ที่อาจช่วยฟื้นฟูการทำงานของสมองในสัตว์แก่ |
| **Prion** | โปรตีนที่พับตัวผิดปกติ สามารถ "ชักชวน" โปรตีนอื่นให้พับผิดตาม |
| **Kuru** | โรคสมองเสื่อมจากไพรออน พบในชนเผ่าฟอร์ ปาปัวนิวกินี |
| **Scrapie** | โรคสมองเสื่อมจากไพรออนในแกะ |
| **CJD (Creutzfeldt-Jakob Disease)** | โรคสมองฟองน้ำในมนุษย์ — เกิดจากไพรออน |
| **BSE (Bovine Spongiform Encephalopathy)** | โรควัวบ้า — เกิดจากไพรออน |
| **Amyloid-beta (Aβ)** | โปรตีนชิ้นเล็กที่สะสมเป็นคราบในสมองผู้ป่วยอัลไซเมอร์ |
| **APP (Amyloid Precursor Protein)** | โปรตีนต้นกำเนิดที่ถูกตัดให้กลายเป็น Aβ |
| **Tau** | โปรตีนในเซลล์ประสาทที่พับผิดแล้วจับกลุ่มเป็นปมในโรคอัลไซเมอร์ |
| **Presenilin** | เอนไซม์ที่ตัด APP — เมื่อกลายพันธุ์ทำให้ผลิต Aβ มากขึ้น |

## 💡 สามข้อคิดที่ควรเอาไปคิดต่อ

1. **ร่างกายเราเหมือนเมืองที่ต้องมีระบบจัดการขยะ** — Proteasome, Lysosome และ Autophagy เป็น "เทศบาล" ที่คอยเก็บขยะ เมื่อระบบเหล่านี้เสื่อมลงตามอายุ "ขยะ" โปรตีนเสียก็สะสมจนเมืองพัง

2. **อัลไซเมอร์เป็นตัวอย่างที่ชัดเจนที่สุดของ "ขยะสะสม"** — คราบ amyloid และปม tau ในสมองคือ "กองขยะ" ที่ระบบกำจัดจัดการไม่ทัน แต่การแค่ "เก็บขยะ" ออกอาจไม่พอ ปัญหาอาจอยู่ที่ระบบทั้งหมดที่ต้องซ่อม

3. **ISRIB และงานวิจัยด้าน ISR เปิดแนวทางใหม่** — แทนที่จะแค่กำจัดโปรตีนเสีย อาจ "ปรับจูน" ระบบตอบสนองความเครียดของเซลล์ให้ทำงานสมดุล เหมือนปรับเครื่องยนต์ไม่ใช่แค่เปลี่ยนน้ำมัน
